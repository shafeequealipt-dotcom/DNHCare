Privacy Policy — DNH Care Digital Assistant

Effective 2026-07-16. This policy covers the DNH Care Digital Assistant software tool only — for how the clinic handles patient health information, please speak with the clinic directly.

Who this tool is for

DNH Care Digital Assistant is a single-tenant internal tool operated by Dr. Nafia's Homoeopathic Medical Centre for its own use only. It has no external users, no public sign-up, and does not process any patient or visitor personal data.

What it accesses, and why

The tool connects to Google's Business Profile API using the https://www.googleapis.com/auth/business.manage scope, authorized by the one Google account that manages the clinic's Business Profile listing. It uses this access only to:

  • Create and remove local posts on the clinic's own Business Profile listing
  • Read the clinic's own listing performance metrics (calls, direction requests, website clicks, search/maps impressions) and the search terms used to find the listing
  • Read the clinic's own review count and average rating (for internal reporting only — the tool never posts replies to reviews)

No data from any other Google Business Profile, and no data belonging to any individual Google user, is ever accessed.

Data storage and retention

Access is via a securely stored OAuth refresh token on a private server operated by the clinic; it is never shared with, or accessible to, any third party. Business Profile performance figures fetched for the weekly report are used transiently to build a summary message and are not stored beyond that. Only the title and Google-assigned reference id of each post the tool creates are kept, so future reports can be matched against them.

No third-party sharing

Data accessed through this integration is never sold, rented, or shared with any third party. It is used solely to operate the clinic's own Business Profile listing and to inform the clinic's own content decisions.
